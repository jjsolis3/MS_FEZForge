GET api/projects/{projectId}/layerAssignment?showRoomDetails={showRoomDetails}

get product assignment info of each Room and Stair

Request Information

URI Parameters

NameDescriptionTypeAdditional information
projectId

Project ID

 string

Required
showRoomDetails

Whether to obtain assignments for doors, windows, elevations, etc., within the room

 boolean

Default value is False

Body Parameters

None.

the layer assignment list.

GET api/projects/length/{pageLength}/page/{pageIndex}?search={search}&distinct={distinct}&orderby={orderby}&isArchived={isArchived}&applicationType={applicationType}
Get paging projects in group

Request Information
URI Parameters
Name	Description	Type	Additional information
pageLength	
the item length of every page. max value is 100.

integer	
Required

pageIndex	
current page index

integer	
Required

search	
fuzzy query project name

string	
None.

distinct	
Count distinct by ProjectId

boolean	
Default value is False

orderby	
"asc" or "desc" by updatedon, default is "asc"

string	
None.

isArchived	
Project is archived or not

boolean	
None.

applicationType	
ApplicationType enum{ FloorCovering, RenoPlan, CounterTop }

string	
None.

Body Parameters
None.
